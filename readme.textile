h1. About

dzone iPhone client, since these are my first iPhone SDK steps, don't expect too much...

See the Screenshots for details.

h2. Current Features for Version 0.0.1

* asynchronously load dzone frontpage feed from "dzone-api.heroku.com":http://dzone-api.heroku.com
* use "Seriously":http://github.com/probablycorey/seriously for JSON parsing
* detail view of feed items

h2. Planned Features

* open article in browser
* vote up, vote down from detail view using personal credentials
* reload button
* personal feed
* personal login credentials and other settings
* store older feed values like twitter app (offline mode?)
